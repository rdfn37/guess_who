# Teste técnico individual - Origem dos nomes

## Projeto

https://gues-who.netlify.app/

Criar aplicação React onde o usuário ao digitar um nome pessoal qualquer em um input de
texto, consiga o retorno formatado no frontend da maior probabilidade da origem do nome
com o nome do país de origem.
Para isso, utilize estas duas fontes, https://api.nationalize.io e
https://gist.github.com/almost/7748738#file-countries-json.

## Adições

Além de utilizar a API nationalize.io para consultar a origem dos nomes, foram utilizadas
também as API's gederize.io e agify.io, que retornam, respectivamente, o gênero e uma idade
estimada. As três API's fazem parte do mesmo pacote e possuem conceitos similares.
